Hosting y Dominio

Abril 13, 2023 Clase 7

Hosting es el espacio en el servidor donde se encuentran los archivos de tu página web y base de datos y que es visitado por los visitantes y los motores de búsqueda. Para que un visitante sea capaz de entrar en una página web, necesitan acceso a una dirección de internet hacia el servidor que contiene los archivos. Esta dirección se llama Protocolo de internet (Internet Protocol) o dirección IP.

Un Dominio es una dirección fácil de recordar para la página web. Seguramente seria fácil escribir “google.com” que escribir 173.194.70.113 en tu navegador. Aunque ambas llevarán al mismo contenido. Tener un dominio para la web no es obligatorio, pero sin él, seguramente seas la única persona que visita la página por dirección IP.
